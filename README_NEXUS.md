# ER Save Manager v2 - Nexus Mods Build

Unofficial Elden Ring save manager with Shadow of the Erdtree support.

## What This Is

This is a local/offline save manager and save editor for Elden Ring. It supports
vanilla `.sl2` saves and Seamless Co-op `.co2` saves.

This Nexus build disables automatic update checks and does not download files at
startup. The app may only open external links when the user manually clicks help
or source links.

## Features

- Shadow of the Erdtree item catalog support.
- `.sl2` and `.co2` support.
- `.sl2` <-> `.co2` conversion.
- Save verification and checksum validation.
- Automatic backups before save-modifying operations.
- Dashboard for current save status.
- Backup Browser.
- Inventory Pro searchable catalog.
- Quantity Editor to view, increase, decrease, or set item quantities.

## Installation

1. Extract the archive anywhere outside the Elden Ring game folder.
2. Run `ER_Save_Manager_v2_Nexus.exe`.
3. Close Elden Ring before editing any save.
4. Set your save folder and SteamID in the app.
5. Run `Verify Current Save` before making changes.

## Safety

- Make a manual backup before using the tool.
- The app creates automatic backups before write operations.
- Do not edit saves while Elden Ring is running.
- Do not upload personal `.sl2` or `.co2` files in bug reports.
- Modified saves may carry online ban risk.
