# Security Policy / Politica De Seguridad

## English

This project handles local Elden Ring save files. Please treat all save files,
Steam IDs, paths, screenshots, logs, and scan reports as potentially personal
data.

### Supported Versions

Only the latest public release is actively supported.

### Reporting A Security Issue

Please open a private report if GitHub Security Advisories are enabled. If they
are not enabled yet, open an issue with a minimal description and avoid posting
private files.

Do not attach:

- `.sl2` or `.co2` save files.
- `ER0000.*` files.
- `data/config.json`.
- Backup folders.
- Scan reports containing personal paths or Steam IDs.

Safe information to include:

- App version or release tag.
- Windows version.
- Whether the save is `.sl2` or `.co2`.
- The exact error message.
- Steps to reproduce using a copied test save.

### Responsible Use

This project is intended for local/offline save management. It does not endorse
online cheating. Modified saves may carry ban risk.

## Espanol

Este proyecto manipula archivos de guardado locales de Elden Ring. Trata saves,
Steam IDs, rutas, capturas, logs y reportes de escaneo como datos
potencialmente personales.

### Versiones Soportadas

Solo se da soporte activo a la release publica mas reciente.

### Reportar Un Problema De Seguridad

Usa un reporte privado si GitHub Security Advisories esta activado. Si aun no
esta activado, abre un issue con una descripcion minima y evita publicar
archivos privados.

No adjuntes:

- Saves `.sl2` o `.co2`.
- Archivos `ER0000.*`.
- `data/config.json`.
- Carpetas de backups.
- Reportes de escaneo con rutas personales o Steam IDs.

Informacion segura para incluir:

- Version de la app o tag de release.
- Version de Windows.
- Si el save es `.sl2` o `.co2`.
- Mensaje exacto de error.
- Pasos para reproducir usando una copia de prueba del save.

### Uso Responsable

Este proyecto esta pensado para administracion local/offline de saves. No
respalda trampas online. Usar saves modificados puede implicar riesgo de ban.
