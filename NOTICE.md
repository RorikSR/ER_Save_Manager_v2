# Notice / Aviso

## English

This repository is an unofficial community project for local save-file management.

Current maintained repository:

- `RorikSR/ER_Save_Manager_v2`: https://github.com/RorikSR/ER_Save_Manager_v2

Before publishing a public release, verify that the code you redistribute is compatible with the licenses and permissions of all upstream projects you used.

Known upstream/reference project:

- `ClayAmore/ER-Save-Editor` is published on GitHub and lists MIT and Apache-2.0 license files.
- Its README asks users to use the editor responsibly, back up saves, and does not endorse online cheating.

Recommended publishing notes:

- Keep attribution to ClayAmore/ER-Save-Editor visible.
- Keep attribution to Ariescyn/EldenRing-Save-Manager visible if this codebase derives from or includes that project.
- Do not upload personal `.sl2` / `.co2` saves.
- Do not upload local configs, backups, or scan reports with personal paths.
- Include a clear warning that online use may carry ban risk.
