## Long Description

```markdown
ER Save Manager v2 is an unofficial local save manager and save editor for Elden Ring, updated for Shadow of the Erdtree.

This Nexus build disables automatic update checks and does not download files at startup.

Features:
- Shadow of the Erdtree item catalog support.
- Vanilla .sl2 and Seamless Co-op .co2 support.
- .sl2 <-> .co2 conversion.
- Save verification and checksum validation.
- Automatic backups before save-modifying operations.
- Dashboard for current save status.
- Backup Browser.
- Inventory Pro searchable catalog.
- Quantity Editor to view, increase, decrease, or set item quantities.

Safety:
- Close Elden Ring before editing saves.
- Keep a manual backup.
- Run Verify Current Save before editing.
- Do not upload personal save files in bug reports.
- Modified saves may carry online ban risk.

Credits:
- Current project: RorikSR/ER_Save_Manager_v2.
- Original reference/upstream credit: ClayAmore/ER-Save-Editor.
- Earlier Python save-manager lineage/reference: Ariescyn/EldenRing-Save-Manager.
- Elden Ring belongs to FromSoftware/Bandai Namco.
- Seamless Co-op belongs to its respective authors.
```
