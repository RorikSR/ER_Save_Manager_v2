# Release Notes - v1.0.0-sote

## English

Initial Shadow of the Erdtree modernization release.

Highlights:

- Support for vanilla `.sl2` saves and Seamless Co-op `.co2` saves.
- Bidirectional `.sl2` <-> `.co2` conversion.
- Checksum validation and recalculation helpers.
- Automatic backups before save-modifying operations.
- Dashboard with current save mode, SteamID, save folder, latest backup, and checksum status.
- Verify Save tool for safer pre-edit checks.
- Backup Browser for reviewing and restoring backups.
- Inventory Pro with searchable catalog fields.
- Quantity Editor to view and adjust current item quantities.
- External JSON item catalogs for base game and Shadow of the Erdtree.
- Real inventory scan reports for catalog maintenance.
- English and Spanish documentation.

Safety notes:

- Always close Elden Ring before editing saves.
- Always keep a manual backup outside the app folder.
- Do not upload personal `.sl2` / `.co2` files to issues.
- Modified saves may carry online ban risk.

Known limitations:

- DLC catalog quality depends on confirmed item IDs.
- Future game patches may require new checksum or inventory validation.
- Licensing should remain conservative until all upstream permissions are fully confirmed.
