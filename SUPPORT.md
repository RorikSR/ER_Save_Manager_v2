# Support / Soporte

## English

Before opening an issue:

1. Close Elden Ring.
2. Make a manual backup of your save.
3. Run `Verify current save` in the app.
4. Check `docs/USAGE.md` and `docs/SAFETY.md`.
5. Confirm you are using the latest release.

When opening an issue, include:

- App version or release tag.
- Windows version.
- Save type: `.sl2` or `.co2`.
- Whether Seamless Co-op is enabled.
- What you clicked before the problem happened.
- The exact error message.

Do not upload personal saves, backups, `config.json`, Steam IDs, or scan
reports with private paths.

## Espanol

Antes de abrir un issue:

1. Cierra Elden Ring.
2. Haz un backup manual de tu save.
3. Ejecuta `Verify current save` en la app.
4. Revisa `docs/es/USAGE.md` y `docs/es/SAFETY.md`.
5. Confirma que usas la release mas reciente.

Al abrir un issue, incluye:

- Version de la app o tag de release.
- Version de Windows.
- Tipo de save: `.sl2` o `.co2`.
- Si Seamless Co-op esta activado.
- Que opcion presionaste antes del problema.
- Mensaje exacto de error.

No subas saves personales, backups, `config.json`, Steam IDs ni reportes de
escaneo con rutas privadas.
